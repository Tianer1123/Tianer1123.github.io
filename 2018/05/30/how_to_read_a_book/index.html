《如何阅读一本书》笔记

发表于 2018-05-30 | | 次阅读
字数统计: 1,348 字 | 阅读时长 ≈ 5 分钟

之前读书没有章法，确切的说自己没学会怎么读一本书，也没有完整的读懂过一本书。《如何阅读一本书》正好教给我如何去读一本自己想读的书。

主动阅读

作者范多伦和艾德列写的这本书内容比较多，结构有点乱。在大框架下穿插讲了一些其他的说明和技巧，例如主动阅读。我感觉可以把他们单独出来当一种技巧来讲，穿插到大框架里反而让人感觉有点乱。

主动阅读总结起来有两点:

  1. 问问题
  2. 做笔记

问问题

  1. 整体来说，这本书在讲什么？
  2. 细部讲什么，怎么说的？
  3. 是否有道理，全部有还是部分有？
  4. 这本书跟我有关系吗？

就是在刨根究底的问一些问题

做笔记

  1. 结构笔记 –> 检视阅读的答案。
    • 这是什么样的书？
    • 整本书在谈什么？
    • 整体框架是什么？
  2. 概念笔记
  3. 辩论笔记

概述阅读的四个层次

阅读分四个层次:基础阅读, 检视阅读, 分析阅读, 主题阅读。

  • 基础阅读: 识字，文字都能读懂。
  • 检视阅读: 在有限时间看完一本书，从表面观察这本书，了解这本书是哪种类型的书。
  • 分析阅读: 真正全面阅读这本书，无限期读懂为止。
  • 主题阅读: 阅读所有与主题相关的书，多用于写论文之类。

每个层次该如何去做

基础阅读

识字和基础的阅读在上小学就已经完成，这个阶段并没有真正的开始阅读，而是做一个准备。

检视阅读

监视阅读分两个步骤（也可以合并成一个步骤）:

  1. 有系统的略读或粗读
    • 看书名，序言
    • 研究目录
    • 索引
    • 作者的介绍
    • 看几个主题
    • 读几段文字
  2. 粗浅的阅读
    从头到尾读一遍这本书，不用全部读懂，无论能读懂多少，总是有收获的。哪怕觉得这本书没有用，不再去读第二遍了。

分析阅读

  1. 分析阅读第一阶段或者说找出这本书在谈些什么东西的四个规则:
    • (1)依照书本的种类与主题作分类。
    • (2)用最简短的句子说出整本书在谈什么。
    • (3)按照顺序关系，列出全书的重要部分。将全书的纲要拟出来以后，再将各个部分的纲要一一列出来。
    • (4)找出作者在问的问题，或者作者想解决的问题。
  2. 分析阅读第二阶段或者说找出一本书到底在说什么的规则(诠释一本书的内容)
    • (5)诠释作者使用的关键字，与作者达成共识。
    • (6)从最重要的句子中抓出作者的重要主旨。
    • (7)找出作者的论述，重新架构这些论述的前因后果，以明白作者的主张。
    • (8)确定作者已经解决了哪些问题，还有哪些问题是未解决的。在未解决的问题中，确定哪些是作者认为自己无法解决的问题。
  3. 分析阅读的第三个阶段，像沟通知识一样评论这本书。
    没读懂不要评论。批评的话要有证据能证明。

主题阅读

主题阅读分两个阶段，一个是准备阶段，一个是主题阅读本身。

观察研究范围: 主题阅读的准备阶段

  1. 针对你要研究的主题，设计一份实验性的书目。可以参考图书馆目录、专家建议与书中的书目索引。
  2. 浏览这份书目上的所有书，确定哪些与你要研究的主题相关，并就你得主题建立起清楚的感念。

主题阅读: 阅读所有第一阶段收集到的书

  1. 浏览所有在第一阶段被认定的与你主题相关的书，找到最相关的章节。
  2. 根据主题创造出一套中立的词汇，带引作者与你达成共识–无论作者是否实际用到这些词汇，所有的作者，或至少绝大部分的作者都可以用这套词汇来诠释。
  3. 建立一个中立的主旨，列出一连串的问题–无论作者是否明白谈过这些问题，所有的作者，或者至少大多数的作者都能要解读为针对这些问题提供了他们的回答。
  4. 界定主要和次要的议题。然后将作者针对各个问题的不同意见整理陈列在各个议题之旁。你要记住，各个作者之间或之中，不见得一定存在着某个议题。有时候你需要针对一些不是作者主要关心范围内的事情，把他的观点解读，才能构建出这种议题。
  5. 分析这些讨论。这的把问题和议题按顺序排列，以求凸显主题。比较有共同性的议题，要放到比较没有共同性议题之前。各个议题之间的关系也要清楚的界定出来。
